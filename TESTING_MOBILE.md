# Тестирование с телефона

## Быстрый старт

1. **Убедитесь, что оба сервера запущены:**
   ```bash
   # Терминал 1 - Backend
   cd backend
   npm run dev
   
   # Терминал 2 - Frontend
   cd frontend
   npm run dev
   ```

2. **Найдите IP адрес вашего компьютера:**
   - Mac/Linux: `ifconfig | grep "inet " | grep -v 127.0.0.1`
   - Windows: `ipconfig`
   - Текущий IP: `192.168.0.147`

3. **Подключите телефон к той же Wi-Fi сети**

4. **Откройте в браузере на телефоне:**
   ```
   http://192.168.0.147:3000
   ```

## Настройка переменных окружения (опционально)

Если нужно использовать конкретный IP адрес, создайте файлы `.env`:

### Backend `.env`:
```
PORT=3001
FRONTEND_URL=*
```

### Frontend `.env.local`:
```
NEXT_PUBLIC_API_URL=http://192.168.0.147:3001/api
```

## Важно

- Оба устройства должны быть в одной Wi-Fi сети
- Брандмауэр может блокировать подключения - разрешите порты 3000 и 3001
- Если IP изменился, обновите `NEXT_PUBLIC_API_URL` в `.env.local`
